# Andromeda Tool Catalog

Curated tools for offline-first AI, local infrastructure, Raspberry Pi deployments, farm/home edge devices, private RAG, voice assistants, and secure self-hosting.

## Foundation and discovery

- [Awesome](https://github.com/sindresorhus/awesome) - master index of curated developer lists.
- [Awesome Selfhosted](https://github.com/awesome-selfhosted/awesome-selfhosted) - self-hosted software map.
- [Awesome Local LLM](https://github.com/rafska/awesome-local-llm) - local LLM ecosystem map.
- [Awesome Raspberry Pi](https://github.com/thibmaek/awesome-raspberry-pi) - Raspberry Pi projects and resources.
- [Awesome Privacy](https://github.com/pluja/awesome-privacy) - privacy-respecting alternatives.
- [Awesome Security](https://github.com/sbilly/awesome-security) - security tooling and resources.
- [Awesome Embedded Rust](https://github.com/rust-embedded/awesome-embedded-rust) - embedded Rust tools.

## Selection rule

A tool belongs in Andromeda when it helps build private, local, offline-capable AI systems. If it depends on cloud APIs by default, it must be clearly labeled as optional and non-default.
